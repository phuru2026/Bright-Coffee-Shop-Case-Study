# Bright-Coffee-Shop-Case-Study
In this assignment I will be conducting a study of Bright Coffee shop.

### Project Description & Raw Data
This folder includes original files that explains what the assignment is about, and the data that was used. Everything is in its original format and nothing was changed.

### Project Planning
In here we find Miro and Gantt Chart, which help understand how the assignment was planned and executed. It also help understand how data was collected, ingested, processed and visualized thoroughly.

### Data Processing
Databricks was used to ingest and process data. I used SQL query to process the data and read the script. In Databricks we get to sort and filter data, step by step until all data was combined into one big table. After getting the big table, data was downloaded and exported to Microsoft Excel, where I used different pivot charts and tables to understand and interpret data visually. I ran multiple charts on one sheet, each with its own heading so it makes it easier to understand.

### Project Presentation
After all the important data have been visualized, it it then exported to Microsoft Powerpoint for a clear and summarized presentation. Here the CEO gets to see the performance of the company and the way forward for the company.
